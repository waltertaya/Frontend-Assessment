# Frontend Design Recreation
A responsive frontend project built using HTML, CSS, JavaScript, and Bootstrap to accurately recreate the provided design layout.

This project was developed as part of a frontend technical assessment, focusing on responsiveness, clean structure, Bootstrap usage, and interactive UI behavior.

## Objective
Recreate the provided UI design while:
- Matching the layout as closely as possible
- Using Bootstrap’s grid system and components
- Making the page fully responsive (desktop, tablet, mobile)
- Adding basic JavaScript interactivity
- Maintaining clean, well-organized code

## Tech stack
- HTML 5
- CSS3
- Javascript (vanilla JS)
- Bootstrap
- Version Control: Git
